카레를 만드는 방법

  1. 야채를 깍둑썰기 해서 준비한다.
  2. 물을 끓이고 카레 가루를 넣는다.
  3. 카레가 끓면 썰어놓은 야채를 투입한다.
  4. 완성되면 밥과 함께 맛있게 먹는다.

웹 관련 용어 정리

HTML
Hyper Text Markup Language의 약자로 웹사이트의 구조를 설계할 때 사용하는 언어이다.
CSS
Cascading Style Sheet의 약자로 마크업 요소에 스타일을 정의할 때 사용하는 언어이다.
